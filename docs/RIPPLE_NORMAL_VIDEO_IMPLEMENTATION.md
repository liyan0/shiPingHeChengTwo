# 普通视频合成水波纹特效实现

**实施日期**: 2026-02-08

---

## 概述

在普通视频合成标签页添加了与带货视频合成页面相同的水波纹特效设置功能。用户现在可以在生成普通视频时应用水波纹特效。

---

## 实现的功能

### 1. UI控件（home_page.py）

在普通视频合成标签页的参数设置面板中添加了"水波纹特效"分组框，包含以下控件：

- **启用水波纹特效** - 复选框，控制是否启用水波纹特效
- **特效区域起始** - 滑块（0-100%），设置水波纹效果开始的垂直位置
- **特效区域结束** - 滑块（0-100%），设置水波纹效果结束的垂直位置
- **波纹强度** - 滑块（1-20），控制波纹的振幅
- **波纹速度** - 滑块（0.1-2.0），控制波纹的移动速度
- **波纹密度** - 滑块（10-100），控制波纹的波长

### 2. 任务管理器更新（normal_video_task_manager.py）

#### 新增参数

在 `NormalVideoTaskManager.__init__()` 中添加了以下参数：

```python
ripple_enabled: bool = False          # 是否启用水波纹
ripple_area_start: int = 50           # 特效区域起始位置（百分比）
ripple_area_end: int = 100            # 特效区域结束位置（百分比）
ripple_amplitude: int = 5             # 波纹强度
ripple_frequency: float = 0.5         # 波纹速度
ripple_wavelength: int = 30           # 波纹密度
```

#### 视频合成逻辑

在 `_generate_single()` 方法中，水波纹特效应用在标题叠加之后：

1. 如果启用水波纹且配置有效（bottom_percent > 0 且 top_percent < 100）
2. 将视频分割为上下两部分
3. 上部分保持不变
4. 下部分应用 geq 滤镜生成水波纹效果
5. 垂直堆叠上下两部分

**FFmpeg 滤镜公式**：
```
geq='X+amplitude*sin((Y+frequency*T)*2*PI/wavelength)':'Cb':'Cr'
```

这个公式创建了随时间垂直移动的水平波纹扭曲效果。

## 使用方法

### 1. 启用水波纹特效

1. 打开应用，切换到"普通视频生成"标签页
2. 在参数设置面板中找到"水波纹特效"分组
3. 勾选"启用水波纹特效"复选框

### 2. 调整参数

- **特效区域起始**: 设置水波纹开始的位置（0%=顶部，100%=底部）
- **特效区域结束**: 设置水波纹结束的位置
- **波纹强度**: 数值越大，波纹扭曲越明显
- **波纹速度**: 数值越大，波纹移动越快
- **波纹密度**: 数值越小，波纹越密集

### 3. 生成视频

点击"开始生成"按钮，水波纹特效将自动应用到生成的视频中。

## 技术细节

### FFmpeg 滤镜链

水波纹特效的实现使用了以下FFmpeg滤镜：

1. **split** - 将视频分割为两个流
2. **crop** - 裁剪上部分和下部分
3. **geq** - 对下部分应用几何方程生成波纹
4. **vstack** - 垂直堆叠上下两部分

### 参数映射

| UI参数 | 内部参数 | 范围 | 默认值 |
|--------|---------|------|--------|
| 特效区域起始 | ripple_area_start | 0-100% | 50% |
| 特效区域结束 | ripple_area_end | 0-100% | 100% |
| 波纹强度 | ripple_amplitude | 1-20 | 5 |
| 波纹速度 | ripple_frequency | 0.1-2.0 | 0.5 |
| 波纹密度 | ripple_wavelength | 10-100 | 30 |

## 已知限制

1. 水波纹特效会增加视频渲染时间（约10-20%）
2. 特效区域必须有效（bottom_percent > 0 且 top_percent < 100）
3. 水波纹仅应用于视频内容，不影响字幕和标题

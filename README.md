## 环境要求

- Python 3.9+
- FFmpeg（需加入系统 PATH）
- Windows 10/11（主要测试平台）
- CUDA（可选，用于本地 Whisper 加速）

## 安装

```bash
# 克隆项目
git clone <repo_url>
cd 带货视频生成

# 安装依赖
pip install -r requirements.txt

# 如需本地语音识别（扒文案功能），额外安装
pip install torch torchvision torchaudio
```

FFmpeg 安装（Windows）：

```bash
# 使用 winget
winget install Gyan.FFmpeg

# 或手动下载 https://ffmpeg.org/download.html，解压后将 bin 目录加入 PATH
```

## 启动

```bash
python main.py
```

## 功能模块详解

### 图片生成

使用即梦 API 批量生成图片。

**前置条件**：在设置页配置即梦 API Key 和 Base URL。

**操作步骤**：
1. 切换到「图片生成」标签页
2. 在提示词框输入图片描述（支持保存为模板）
3. 设置生成数量、图片比例（1:1 / 16:9 / 9:16 等）、分辨率（1k/2k/4k）
4. 点击「开始生成」
5. 生成完成后图片保存至 `input/视频图片素材文件夹/`

**支持模型**：jimeng-4.5、jimeng-4.1、jimeng-4.0、jimeng-3.x、jimeng-2.x、jimeng-xl-pro 等

---

### 视频生成

使用视频生成 API（兼容 veo_3_1 等模型）将图片转为视频。

**前置条件**：配置视频 API Key；`input/视频图片素材文件夹/` 中有图片素材。

**操作步骤**：
1. 切换到「视频生成」标签页
2. 设置生成数量、视频时长（4/8/12 秒）、分辨率（720x1280 / 1280x720 / 1080x1080）
3. 可选填运动提示词
4. 点击「开始生成」
5. 视频下载至 `input/视频素材/`

**参数说明**：
- 并发数：同时提交的任务数，建议 3-5，过高可能触发 API 限流
- 最大重试：单个任务失败后的重试次数

---

### 视频文案生成

调用 LLM API 批量生成带货视频文案。

**操作步骤**：
1. 切换到「视频文案」标签页
2. 编写或选择文案生成提示词模板
3. 设置生成数量和并发数
4. 点击「开始生成」
5. 文案保存至 `input/视频文案/` 对应子目录

**提示词模板**：支持保存多个模板，下次直接选用。

---

### 图片识别

使用 Gemini 等视觉模型识别商品图片，自动生成商品描述文案。

**前置条件**：`input/商品图片/<商品名>/` 中有商品图片。

**操作步骤**：
1. 切换到「图片识别」标签页
2. 选择要识别的商品文件夹（可多选）
3. 编写识别提示词（如：请描述这个商品的特点、卖点）
4. 点击「开始识别」
5. 识别结果保存至 `input/商品文案/<商品名>/`

---

### 文案合并

将视频文案与商品文案合并，生成完整的带货文案。

**前置条件**：
- `input/视频文案/` 中有视频文案
- `input/商品文案/<商品名>/` 中有商品文案

**操作步骤**：
1. 切换到「文案合并」标签页
2. 选择要合并的商品文件夹
3. 设置合并数量和提示词
4. 点击「开始合并」
5. 合并结果保存至 `input/合并文案/<商品名>/`

---

### 语音生成

将文案批量转换为语音，支持带货模式和流量模式。

**前置条件**：配置 TTS API Key；对应目录中有文案 txt 文件。

**操作步骤**：
1. 切换到「语音生成」标签页
2. 选择模式：
   - **带货模式**：读取 `input/合并文案/<商品名>/`，输出至 `input/视频配音/<商品名>/`
   - **流量模式**：读取 `input/视频文案/流量文案/`，输出至 `input/视频配音/流量语音/`
   - **引导模式**：读取 `input/视频文案/引导文案/`，输出至 `input/视频配音/引导语音/`
3. 选择商品文件夹（带货模式）
4. 设置生成数量、音色、语速
5. 可选开启：
   - **同步生成字幕**：语音生成后自动生成 SRT 字幕
   - **识别商品时间**：自动识别语音中介绍商品的时间段（用于视频合成时精准插入商品素材）
   - **字幕纠错**：对生成的字幕进行 AI 纠错
6. 点击「开始生成」

**音色选项**（gpt-4o-mini-tts）：alloy、echo、fable、onyx、nova、shimmer 等

**字幕生成方式**：
- API 模式：调用 Whisper API，速度快，需联网
- 本地模式：使用本地 faster-whisper 模型，支持 small/medium，可选 CPU/CUDA

---

### 字幕生成

对已有音频文件单独生成字幕。

**操作步骤**：
1. 切换到「字幕生成」标签页
2. 选择音频文件目录
3. 选择生成方式（API / 本地）
4. 设置语言（默认中文）、每段最大字数
5. 点击「开始生成」，SRT 文件保存在音频同目录

---

### 带货视频合成

核心功能。将语音、视频素材、商品素材、BGM 合成为完整带货视频。

**前置条件**：
- `input/视频配音/<商品名>/` 中有语音 mp3 和对应 srt 字幕
- `input/视频素材/` 中有背景视频素材
- `input/商品/<商品名>/` 中有商品视频（可选）
- `input/商品图片/<商品名>/` 中有商品图片（可选）
- `input/背景音乐/` 中有 BGM（可选）

**操作步骤**：
1. 切换到「视频合成」标签页
2. 选择商品文件夹
3. 配置合成参数：
   - **分辨率**：1080p竖屏（1080x1920）/ 720p竖屏 / 1080p横屏 等
   - **片段时长**：每段背景视频的时长范围（秒），支持随机范围
   - **生成数量**：批量生成的视频数量
   - **并发数**：同时处理的任务数（建议 2-3，FFmpeg 较占资源）
   - **商品视频数量**：每个视频中插入的商品视频片段数
   - **商品图片数量**：每个视频中插入的商品图片数量
   - **商品图片时长**：每张商品图片的展示时长范围（秒）
   - **BGM 音量**：背景音乐音量（0-100）
   - **配音音量**：语音音量（0-100）
   - **优先使用视频**：有商品视频时优先使用视频而非图片
   - **叠加模式**：商品素材以叠加（画中画）方式显示，而非替换背景
4. 配置特效（可选，见下方特效配置）
5. 点击「开始合成」
6. 输出至 `output/<商品名>/`

**商品素材插入逻辑**：
- 根据语音中识别到的商品时间段，在对应时间点插入商品视频/图片
- 未识别到商品时间时，均匀分布在视频中
- 商品图片使用 contain 缩放（保持原始宽高比，透明填充），并附带缓慢缩放平移动画

### 普通视频生成

不含商品素材，仅用语音 + 背景视频 + BGM 合成流量视频或引导视频。

**操作步骤**：
1. 切换到「普通视频」标签页
2. 勾选视频类型：流量视频 / 引导视频（可同时勾选）
3. 配置分辨率、片段时长、数量、音量等参数
4. 点击「开始生成」
5. 输出至 `output/流量视频/` 或 `output/引导视频/`

---

### 扒文案

从本地视频文件中提取语音并转为文案，使用本地 faster-whisper 模型。

**操作步骤**：
1. 切换到「扒文案」标签页
2. 选择包含视频的目录
3. 选择 Whisper 模型（small 速度快，medium 精度高）
4. 选择设备（CPU / CUDA）
5. 点击「开始提取」
6. 文案 txt 保存在视频同目录

**本地模型路径**：
- `models/faster-whisper-small/`
- `models/faster-whisper-medium/`

如目录不存在，首次运行会自动从 HuggingFace 下载。

---

### 文案改写

对已有文案进行 AI 改写，生成多个版本。

**操作步骤**：
1. 切换到「文案改写」标签页
2. 选择文案目录
3. 编写改写提示词
4. 设置每篇生成版本数和并发数
5. 点击「开始改写」
6. 改写结果保存在原目录，文件名加 `_1`、`_2` 等后缀

## 视频特效配置

在视频合成和普通视频生成页面，点击对应按钮打开特效设置对话框。

### 水波纹特效

为视频底部添加水面倒影波纹效果。

| 参数 | 说明 | 范围 |
|------|------|------|
| 倒影比例 | 倒影区域占视频高度的比例 | 0.1 - 0.5 |
| 振幅 | 波纹幅度 | 0.1 - 2.0 |
| 频率 | 波纹密度 | 0.1 - 2.0 |
| 速度 | 波纹流动速度 | 0.1 - 2.0 |
| 不透明度 | 倒影透明度 | 0.1 - 1.0 |
| 色调强度 | 倒影色调偏移 | 0.0 - 1.0 |

### 模糊边框

为竖屏视频添加模糊背景边框，适合横版素材转竖屏。

| 参数 | 说明 | 范围 |
|------|------|------|
| 边框宽度 | 边框占视频宽度的百分比 | 1% - 20% |
| 模糊强度 | 高斯模糊 sigma 值 | 5 - 100 |

### 画中画（PIP）

在视频中叠加一个圆形画中画窗口。

| 参数 | 说明 | 范围 |
|------|------|------|
| 大小 | 圆形直径占短边的百分比 | 10% - 60% |
| 水平位置 | 0% = 左，100% = 右 | 0 - 100 |
| 垂直位置 | 0% = 上，100% = 下 | 0 - 100 |
| 边框粗细 | 圆形边框像素宽度 | 0 - 10 |
| 边框颜色 | 十六进制颜色值 | - |

### 叠加素材

在视频上叠加特效素材（扫光、飘落粒子等）。

**素材准备**：将特效视频（带透明通道的 webm/mov）放入 `input/叠加素材/<子文件夹名>/`。

在叠加素材设置中，勾选要使用的子文件夹，并设置不透明度（0-100）。

## 素材准备指南

### 背景视频素材

- 格式：mp4、mov、avi、mkv、wmv、flv
- 建议分辨率：1080x1920（竖屏）或更高
- 建议时长：10-60 秒/段，数量越多随机性越好
- 放置路径：`input/视频素材/`

### 商品视频

- 格式同上
- 建议时长：10-60 秒
- 按商品名建子文件夹：`input/商品/<商品名>/`

### 商品图片

- 格式：jpg、jpeg、png、bmp、webp
- 支持任意宽高比（工具会自动 contain 缩放，保持原比例，透明填充）
- 按商品名建子文件夹：`input/商品图片/<商品名>/`

### 背景音乐

- 格式：mp3、wav、aac、m4a、flac
- 建议时长：2 分钟以上（不足时自动循环）
- 放置路径：`input/背景音乐/`

### 文案文件

- 格式：UTF-8 编码的 txt 文件
- 每个文件对应一条文案
- 文件名即为视频标题（如开启标题功能）

## 常见问题

**Q: FFmpeg 报错 / 找不到 ffmpeg**

确认 FFmpeg 已安装并加入 PATH：
```bash
ffmpeg -version
```

**Q: 本地 Whisper 模型下载慢**

可手动下载模型文件放入 `models/faster-whisper-small/` 或 `models/faster-whisper-medium/`，模型文件从 HuggingFace 的 `Systran/faster-whisper-small` 仓库获取。

**Q: 视频合成时商品图片变形**

已修复（v1.x+）。工具现在使用 contain 缩放模式，非 9:16 比例的商品图片会保持原始宽高比，两侧透明填充。

**Q: 字幕乱码**

确认文案 txt 文件为 UTF-8 编码。可用记事本另存为 UTF-8 格式。

**Q: API 请求失败 / 超时**

- 检查 API Key 是否正确
- 检查 Base URL 是否可访问
- 适当降低并发数
- 增加最大重试次数

**Q: 生成的视频没有声音**

检查语音文件是否存在于 `input/视频配音/<商品名>/`，且文件名与文案文件名对应。

**Q: Windows 启动时 DLL 冲突**

项目已在 `main.py` 中预加载 torch 以规避此问题。如仍出现，尝试：
```bash
set KMP_DUPLICATE_LIB_OK=TRUE
python main.py
```
